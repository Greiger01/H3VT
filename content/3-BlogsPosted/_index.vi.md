Tại đây sẽ là phần liệt kê, giới thiệu các blogs mà các bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). Ví dụ:

###  [Blog 1 - Tối Ưu Khả Năng Phục Hồi Ứng Dụng Toàn Cầu Với AWS Global Accelerator](3.1-Blog1/)
Bài blog nói về giải pháp AWS Global Accelerator giúp tối ưu hiệu năng mạng và tự động chuyển vùng khi có sự cố (Failover) gần như ngay lập tức cho các ứng dụng toàn cầu.

###  [Blog 2 - Physics on AWS: Tối ưu hiệu suất tuabin gió bằng OpenFAST trong digital twin](3.2-Blog2/)
Bài blog phân tích cách AWS kết hợp dữ liệu IoT thời gian thực, Amazon Timestream, AWS Lambda, Amazon S3, AWS Fargate và Amazon Managed Grafana để xây dựng digital twin cho tuabin gió, hỗ trợ phát hiện bất thường và tối ưu hiệu suất vận hành.

###  [Blog 3 - Cách Sonar xây dựng unified API trên AWS](3.3-Blog3/)
Bài blog giới thiệu cách Sonar kết hợp Elastic Load Balancing, AWS PrivateLink và Amazon API Gateway để xây dựng một unified API endpoint bảo mật, dễ vận hành và phù hợp với kiến trúc nhiều domain độc lập.
